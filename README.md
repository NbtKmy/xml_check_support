# XML Check Support

OAI-PMH形式のXMLファイルを目視チェックするためのブラウザベースのサポートアプリです。
データは外部サーバへ送信されず、すべてブラウザ内で処理されます。

## 起動方法

**`index.html` をダブルクリックして、ブラウザで開いてください。**

それだけで使えます。インストールやサーバのセットアップは不要です。

> ターミナルを使う場合は `open index.html` でも起動できます。

## 使い方

### 1. XMLファイルを開く

- ヘッダーの **「XMLを開く」** ボタンをクリック
- または、ファイルをウィンドウ上に **ドラッグ＆ドロップ**

### 2. レコードを閲覧する（Col 1 → Col 2）

- 左のリストからレコードをクリックすると、右の詳細パネルに全フィールドが表示されます
- キーボードの **↑ / ↓** でレコードを順番に移動できます

### 3. 検索する

- ヘッダーの検索欄にキーワードを入力すると、タイトル・作成者・ID・説明・主題などを横断して絞り込みます
- ヒット箇所は黄色でハイライトされます
- キーボードの **`/`** キーで検索欄にフォーカスできます

### 4. フィルターで絞り込む

- **種別**（写真・図書など）と **Set**（書誌セット）のプルダウンで組み合わせ絞り込みができます
- 検索キーワードとの併用も可能です

### 5. フィールドブラウザを使う（Col 3）

詳細パネルのフィールドラベル（`description`、`作成者`、`件名` など）にホバーすると **≡** アイコンが表示され、クリックすると右側にフィールドブラウザが開きます。

**フィールドブラウザでできること：**

| 操作 | 内容 |
|------|------|
| フィールドラベルをクリック | 全レコードのその要素の値一覧を表示（出現件数付き） |
| 値をクリック | その値を持つレコードをリストで 🟡 ハイライト |
| 同じ値を再クリック | ハイライト解除 |
| ブラウザ内の検索欄 | 値リストをさらに絞り込む |
| 同じフィールドラベルを再クリック / × ボタン | フィールドブラウザを閉じる |

**例：** `description` をクリックすると「拠点名: 正門」「写真種別: ライカ」などの値が全レコード分一覧表示され、どんな値が何件あるか一目でわかります。

### 6. 生のXMLを確認する

詳細パネル下部の **「生のXMLを表示」** をクリックすると、そのレコードの整形済みXMLを確認できます。

### 7. 外部リンクを開く

`参照先` に表示されたリンクはクリックで外部サイトを新しいタブで開きます。

## キーボードショートカット

| キー | 動作 |
|------|------|
| `/` | 検索欄にフォーカス |
| `↑` / `↓` | レコードリストを移動 |
| `Esc` | 検索欄からフォーカスを外す / フィールドブラウザを閉じる |

---

## 対応フォーマット

OAI-PMH + Dublin Core (dcndl) 形式のXMLファイルを想定しています。
国立国会図書館サーチ（NDLサーチ）のOAI-PMH APIレスポンスをそのまま読み込めます。

主に以下の要素を抽出・表示します：

- `dcterms:title` / `dc:title` — タイトル
- `foaf:name` / `dc:creator` — 作成者
- `dcterms:date` / `dcterms:issued` — 日付
- `dcterms:description` — 説明
- `dcterms:subject` — 主題
- `dcndl:materialType` — 資料種別
- `dcterms:spatial` / `dcterms:temporal` — 場所・時間
- `rdfs:seeAlso` — 外部リンク
- `header > identifier` / `datestamp` / `setSpec` — 管理情報
